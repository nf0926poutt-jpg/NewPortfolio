# マークダウン形式のファイルの書き方

「マークダウン形式のファイル」とは、（装飾されていないただの文字情報）で記述され、拡張子が .md であるファイルのことです。
これは、通常の文章の中に、簡単な記号を使って見出しやリストなどの構造を定義するためのファイル形式です。  

マークダウン形式で書かれたファイルは、HTML形式に変換されて表示されます。  

Githubの「README.md」などはこの形で記述されることが多く、簡易的なドキュメントであれば、HTMLよりも効率的にドキュメントの作成ができます。

```
# メインタイトル
```

```
## サブタイトル
```
